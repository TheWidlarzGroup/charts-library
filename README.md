---

### Installation:

- [ ] Clone the repo
- [ ] Run `yarn` from the root
- [ ] Run `cd ios && pod install && cd ..`
- [ ] Run the metro with `yarn start`
- [ ] Open the app either in a simulator with `yarn ios` / `yarn android` or on a physical device
- [ ] Use the examples in your project :smile:
